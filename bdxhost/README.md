# bdx Electron Window Host

Requires dotnetcore 2.1 SDK, yarn and node >= 9

## Setup (yarn required)

```
yarn install
yarn run build-native
```

## Run locally (dev)

```
yarn run dev
```

## Run locally (prod)

```
yarn start
```

## Build One Click Installer

```
yarn run package
```
